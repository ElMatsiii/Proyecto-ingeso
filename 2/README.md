# Click & Buy — Tienda de Cartas Pokémon

**Click & Buy** es una aplicación web que simula un e-commerce para la compra de **cartas Pokémon individuales**.  
El proyecto está desarrollado con **HTML**, **CSS**, **TypeScript** y utiliza el **SDK oficial de TCGdex** para obtener información real de cartas Pokémon TCG.

---

## Características principales

- **Catálogo dinámico:** las cartas se obtienen desde la API oficial de TCGdex.
- **Carrito persistente:** las cartas añadidas se guardan en `localStorage`.
- **Vista de producto:** permite mostrar detalles de cada carta (imagen, tipo, rareza, precio).
- **Formulario de contacto:** simula el envío de mensajes desde la sección *Acerca de*.
